# Projekt infrastruktury LAN z segmentacją VLAN

Projekt przedstawia koncepcję sieci LAN dla budynku wielokondygnacyjnego z wykorzystaniem VLAN, adresacji IPv4 oraz centralnego zarządzania MDF/IDF.

## Zakres projektu

- projekt topologii sieci,
- segmentacja VLAN,
- adresacja IPv4,
- konfiguracja DHCP i DNS,
- projekt MDF/IDF,
- okablowanie strukturalne,
- model sieci w Cisco Packet Tracer.

## Technologie

- VLAN
- IPv4
- DHCP
- DNS
- Cisco Packet Tracer
- LAN/WLAN
- MDF/IDF
